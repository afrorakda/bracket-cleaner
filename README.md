# Bracket Cleaner (Fridge Combo #43)

A hyper-minimalist web tool to instantly remove brackets or the text inside them. Designed for clean, distraction-free text processing.

## Core Features
* **Two Modes**:
    * **Delete Content**: Removes brackets and everything inside them (e.g., `Text (info)` → `Text`).
    * **Brackets Only**: Removes only the bracket symbols, keeping the text inside (e.g., `Text (info)` → `Text info`).
* **Auto-Copy**: Click the "Clean Text" button to process and automatically copy the result to your clipboard.
* **Comprehensive Support**: Handles 14 types of brackets including `()`, `[]`, `{}`, `「」`, `【】`, `<>`, and more (full-width & half-width).

## Design Philosophy
* **1 Tool / 1 Action**: Focused entirely on bracket removal. No extra menus or settings.
* **Privacy First**: No login, no tracking, and no ads. All processing happens locally in your browser.
* **Performance**: Lightweight and optimized for both mobile and low-spec PCs.
* **Consistent UI**: Follows the "Fridge Combo" design system (White/Grey/333 Black with 15px rounded corners).

## Privacy Policy
* **Browser-Only**: Your text never leaves your device. Data is processed locally in the browser.
* **No Tracking**: No cookies, no analytics, and no data collection.
